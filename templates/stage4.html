Line Tracing Algorithm

The red circle is the light sensor, and takes in a certain input.
If the robot is more on the black line than the white side, the value of the light sensor is lower (i.e. darker), and vice versa.
The goal is to keep it on the centre at all times to keep it straight.

The robot moves using the motor A (left motor) and motor C (right motor).
If they both move at the same speed, the robot moves straight.
If A (left motor) moves faster, the robot turns right, and vice versa.
